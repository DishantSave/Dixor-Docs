UUID Version 7 Validation (Uuid7.IsUuid7 & Uuid7.IsValid)

📚 Navigation: Home > Dixor.Identity > Validation

Many developers assume that all .NET Guid structures are interchangeable. However, legacy Guid.NewGuid() generates random Version 4 values, which lack timestamps and cause severe index page splits. Furthermore, advanced Dixor.Identity operations—such as Timestamp Extraction—strictly require genuine RFC 9562 UUID Version 7 identifiers.

Dixor.Identity exposes two ultra-fast, zero-allocation methods to verify identifier compliance:

using Dixor.Identity.UUID7;

// 1. Verify an already instantiated Guid struct
bool isGenuineV7 = Uuid7.IsUuid7(myGuid);

// 2. Validate incoming text strings (e.g., HTTP headers, query parameters)
bool isValidString = Uuid7.IsValid("ORD-0197d8ef-f4c6-7a35-99e3-1d4a7e3c8e10");

Bitwise Validation Anatomy

To verify compliance in constant O(1) time without heap allocation, Uuid7.IsUuid7(Guid) inspects the High Nibble (most significant 4 bits) of Byte 6 inside the 16-byte UUID layout:

Byte 6 Layout:
+---+---+---+---+---+---+---+---+
| V | V | V | V | x | x | x | x |
+---+---+---+---+---+---+---+---+
└───────┬───────┘
        ▼
 Version Bits (Must equal 0111 / 0x7)

By allocating a 16-byte buffer on the execution stack (stackalloc byte[16]) and right-shifting Byte 6 by four positions (bytes[6] >> 4), the validator verifies RFC 9562 Version 7 adherence in sub-nanosecond execution time.

API Overloads & Input Handling

Method Accepts Behavior Typical Use Case
Uuid7.IsUuid7(Guid) Guid struct Verifies if the in-memory GUID has Version 7 bits (0x7). Validating internal domain models and database records.
Uuid7.IsValid(string) string text Parses raw text via Uuid7Parser.TryParse and verifies Version 7 bits. Protecting REST API endpoints, webhooks, and CSV imports.

Securing API Boundaries

Always validate external identifiers at your REST controllers or Minimal API endpoints before passing them to domain logic:

app.MapGet("/api/orders/{id}", (string id) =>
{
    // Rejects non-GUID strings AND legacy UUIDv4 identifiers instantly
    if (!Uuid7.IsValid(id))
    {
        return Results.BadRequest("Identifier must be a valid RFC 9562 UUID Version 7.");
    }

    // Proceed safely with valid V7 processing...
    return Results.Ok();
});

Performance Characteristics

  • Time Complexity: O(1) Constant Time.
  • Space Complexity: O(1) Zero Heap Allocation (stackalloc byte[16]).
