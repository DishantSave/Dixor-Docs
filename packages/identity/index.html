Dixor.Identity Package

Welcome to Dixor.Identity, the core library for allocation-conscious, high-performance identifier generation and encoding within the Dixor ecosystem.

Package Modules & Identifiers

  • UUIDv7: High-performance, time-sortable 128-bit identifier generation built for optimized database indexing.
  • ULID: Specification-compliant 128-bit Universally Unique Lexicographically Sortable Identifiers combining millisecond timestamps with Crockford Base32 encoding.
  • Base36: Compact alphanumeric encoding utilities for case-insensitive identifier representations.
  • Base62: URL-safe Base62 encoding components optimized for dense, short string representations.
  • Hexadecimal: Direct hex formatting and parsing tools for low-level binary inspection and interoperability.
  • NanoId: Compact, URL-friendly, secure string identifier generation designed as a compact alternative to UUIDs.
  • Numeric Code: Specialized numeric-only code generation tools for pin codes, verification tokens, and sequential identifiers.
  • Sequence: High-throughput, thread-safe sequential ID generation for ordered data streams.
  • Short Identifier: Optimized short identifier generators for readable, compact application references.

Getting Started

To install the package, add it to your project via NuGet:

dotnet add package Dixor.Identity

Interactive Studio & Exploration

You can visually inspect, test generation patterns, and explore real-time bit layouts and encodings interactively via Dixor Studio:

  • UUIDv7 Studio: Test standard/monotonic generation and inspect UUIDv7 anatomical structures.
  • ULID Studio: Explore Crockford Base32 encoding, timestamp extraction, and sorting mechanics.
