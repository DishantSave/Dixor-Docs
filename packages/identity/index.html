# UUIDv7 Implementation

The **UUIDv7** module provides a zero-allocation, thread-safe implementation of RFC 9562 Version 7 identifiers. These are designed to replace legacy random GUIDs, providing sequential, time-based ordering for optimized database indexing.

## Key Features
* **Sequential Sorting:** Naturally ordered B-Tree friendly IDs.
* **Zero Allocation:** Built for high-throughput .NET 10+ workloads.
* **Metadata Embedded:** Easily extract creation timestamps from the ID payload.

## Core Guides
* [Overview](UUID7_Overview.md) - Understanding the anatomy of a v7 ID.
* [Generation](UUID7_Generation.md) - Standard and monotonic generation patterns.
