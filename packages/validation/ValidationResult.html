ValidationResult Reference

The ValidationResult<TError> is a generic, high-performance record struct used throughout the Dixor.Validation library to return the outcome of any validation operation.

Overview

By using a record struct, this type provides value-type semantics (avoiding heap allocations) while maintaining immutability and easy equality checks. It forces callers to explicitly handle success or failure states via the IsValid property.

Usage

The result is designed for simple, idiomatic usage within service methods:

public ValidationResult<EmailValidationError> Validate(string email)
{
    if (string.IsNullOrEmpty(email))
        return ValidationResult<EmailValidationError>.Failure(EmailValidationError.Empty);

    return ValidationResult<EmailValidationError>.Success();
}

// Consumer consumption
var result = validator.Validate("test@example.com");
if (result.IsValid) 
{
    // Proceed...
} 
else 
{
    Console.WriteLine($"Error: {result.Error}");
}
