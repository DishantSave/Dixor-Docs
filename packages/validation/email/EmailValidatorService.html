EmailValidatorService Reference

The EmailValidatorService is the internal orchestration layer for the Dixor.Validation.Email library. It acts as the primary bridge between the public EmailValidator API and the specialized internal services.

Overview

This service does not contain validation logic itself. Instead, it coordinates calls to IEmailParser and IEmailNormalizer. This ensures a single, consistent execution pipeline for all validation, normalization, and inspection tasks.

Responsibilities

  • Validation Coordination: Executes the parsing pipeline via IEmailParser.
  • Normalization Gateway: Exposes IEmailNormalizer functionality for consistent string representation.
  • Component Access: Simplifies the retrieval of specific email parts (LocalPart, Domain, Subdomain) by wrapping EmailInfo extraction.
  • Domain Logic: Implements domain matching and TLD checks using case-insensitive comparisons.
  • Comparison Logic: Facilitates equality checks by normalizing both inputs before comparison.
