Remarks

EmailValidationError provides a strongly typed set of error codes returned by the email validation pipeline. Each member represents a specific validation failure encountered while parsing or validating an email address.

When validation succeeds, None is returned. Otherwise, one of the remaining members identifies the exact reason the email address failed validation.

These values are intended to be consumed by ValidationResult<TError> and other library APIs to provide deterministic, allocation-free error reporting.
