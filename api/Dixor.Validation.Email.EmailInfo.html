Remarks

EmailInfo exposes the individual components of a successfully parsed email address.

Instances of this type are created internally by the email parser and are reused by validation and inspection APIs to avoid repeatedly parsing the same email address.

This type is immutable and contains only structural information that can be directly derived from the email address itself.
