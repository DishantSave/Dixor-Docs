Remarks

Ulid serves as the central entry point for all 128-bit Universally Unique Lexicographically Sortable Identifier operations within the Dixor.Identity ecosystem.

Core capabilities exposed by this facade include:

  • Standard Generation: Creates timestamp-based ULIDs using cryptographically secure randomness (Generate()).
  • Monotonic Sequencing: Generates strictly increasing ULIDs during same-millisecond bursts (GenerateMonotonic()).
  • High-Efficiency Batching: Provides lazy generation of multiple ULIDs for bulk operations (Generate(int)).
  • Binary Conversion: Converts between canonical ULID strings and 16-byte representations (Parse(string), Format(ReadOnlySpan<byte>)).
  • Validation: Performs fast verification of ULID string representations (IsValid(string)).
  • Timestamp Inspection: Extracts embedded creation timestamps without additional metadata storage (GetTimestamp(string)).

ULIDs generated by this class follow the ULID specification: a 48-bit millisecond timestamp combined with 80 bits of randomness and encoded using Crockford Base32 into a 26-character identifier.
