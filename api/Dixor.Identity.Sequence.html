Remarks

Sequence is the primary entry point for creating sequential integer generators in the Dixor.Identity library.

Sequences generate monotonically increasing integer values with configurable starting values and increments. They are suitable for generating sequential identifiers such as invoice numbers, order numbers, ticket numbers and other application-specific sequences.

Every sequence instance maintains its own independent state and is safe for concurrent use from multiple threads and asynchronous operations.
