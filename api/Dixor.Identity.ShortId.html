Remarks

ShortId is the primary entry point for generating short, human-friendly identifiers in the Dixor.Identity library.

Short identifiers are generated using a carefully selected character set that excludes visually ambiguous characters, improving readability and reducing the likelihood of transcription errors when identifiers are manually entered or communicated.

All identifiers are generated using a cryptographically secure random number generator to ensure high entropy, uniqueness and unpredictability.
