Defines a contract for a thread-safe integer sequence.

public interface ISequenceService

Remarks

Implementations of ISequenceService are responsible for generating sequential integer values.

Each sequence instance maintains its own independent state, allowing multiple sequences to operate concurrently without interfering with one another.
